# IntelliMaint 智能维护平台
## 项目汇报

---

### 一、项目概述

**IntelliMaint** 是一套面向工业场景的 **AI 预测性维护平台**，实现对工业设备的实时监控、智能告警、故障预测，帮助企业从"被动维修"转向"主动预防"，显著降低设备停机损失。

|------|------|
| 项目版本 | v56.1 |
| 代码规模 | 15,000+ 行 |
| 技术栈 | .NET 8 + React 18 |
| 开发周期 | 持续迭代中 |

---

### 二、核心功能亮点

#### 1. 实时数据采集与监控
- **毫秒级数据采集**：支持 OPC UA、LibPlcTag 等主流工业协议
- **实时数据推送**：基于 SignalR WebSocket，延迟 < 50ms
- **多设备并发**：单节点支持 1000+ 标签点位采集

#### 2. 智能告警引擎
- **多维度告警规则**：
  - 阈值告警（高/低限值）
  - 离线检测（设备掉线自动感知）
  - 变化率告警（RoC - 异常突变检测）
  - **波动告警**（标准差监测，识别异常抖动）
- **告警聚合**：相同设备/规则的告警自动归组，避免告警风暴
- **四级严重度**：Info → Warning → Alarm → Critical

#### 3. 数据分析能力
- **历史数据查询**：支持时间范围、设备、标签多维筛选
- **周期分析**：识别设备运行周期规律
- **数据导出**：支持 CSV、Excel 格式导出

#### 4. 安全与权限
- **JWT 认证**：15分钟 Access Token + 7天 Refresh Token
- **RBAC 三级权限**：Admin / Operator / Viewer
- **完整审计日志**：所有关键操作可追溯

### 三、技术架构亮点

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (React)                          │
│    实时监控面板 │ 数据查询 │ 告警管理 │ 设备配置 │ 系统管理    │
└─────────────────────────────────────────────────────────────┘
                              │ HTTP/WebSocket
┌─────────────────────────────────────────────────────────────┐
│                      API 网关 (.NET 8)                       │
│         Minimal API │ SignalR Hub │ JWT Auth                │
└─────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────┐
│                      业务服务层                              │
│    告警引擎 │ 健康评估 │ 数据聚合 │ 离线检测 │ 波动监测        │
└─────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────┐
│                      数据采集层                              │
│           OPC UA │ LibPlcTag │ Modbus (规划中)              │
└─────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────┐
│                     工业设备 / PLC                           │
└─────────────────────────────────────────────────────────────┘
```

**架构优势**：
- **分层解耦**：Core / Infrastructure / Application / Host 四层架构
- **高性能**：API P95 响应时间 < 100ms
- **易扩展**：新增协议只需实现 `IProtocolConnector` 接口
- **云原生就绪**：支持 Docker 容器化部署
